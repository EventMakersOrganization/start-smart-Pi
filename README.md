# Context-Aware Adaptive Learning Platform

## Overview
This project was developed as part of the PIDEV (Projet d’Intégration et de Développement) – 4th Year Engineering Program at Esprit School of Engineering (Academic Year 2025–2026).

The platform aims to enhance the first-year academic experience by providing a context-aware adaptive learning system.  
It personalizes educational content based on user behavior, progress, and learning patterns to improve engagement and academic performance.

---

## Features
- User authentication and role-based access
- Student and Admin dashboards
- Context-aware content adaptation
- Personalized learning recommendations
- Progress tracking and performance analytics
- Secure RESTful API integration
- Responsive UI design

---

## Tech Stack

### Frontend
- Angular
- TypeScript
- Tailwind CSS

### Backend
- NestJS
- Node.js
- MongoDB
- Mongoose

---

## Architecture
The application follows a modern full-stack architecture:

- Frontend: Angular Single Page Application (SPA)
- Backend: NestJS REST API
- Database: MongoDB NoSQL database
- Communication: HTTP REST APIs
- Authentication: JWT-based authentication system

The project follows modular architecture principles with separation of concerns between controllers, services, and data models.
